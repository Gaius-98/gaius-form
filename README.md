# scaling-dollop
### 对于一些简单的页面或表单采用配置的方式自动生成，提高开发效率。
 #### 配置表单
 基于element-plus组件生成对应的vue2/vue3的表单组件代码。
 简单，方便。
 #### 配置页面
 通过拖拽、配置组件的方式生成对应的页面。
